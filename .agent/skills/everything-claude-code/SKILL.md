# Everything Claude Code - Capability Directory

## 📋 Overview

Everything Claude Code is the central hub for discovering and integrating new capabilities into the Antigravity Kit. It serves as a directory for:
- **Plugins**: Third-party extensions.
- **Skills**: Domain-specific knowledge modules.
- **Resources**: "Awesome" lists and best-practice guides.

## 🛠️ Usage

### 1. Discovery
- When a task requires a capability not found in the current `.agent/skills` folder, search for external "Awesome" resources or community skills.
- Reference: `https://github.com/Stitch-AI-LLC/everything-claude-code` (Conceptual)

### 2. Integration
- New skills should follow the **Antigravity Protocol** (SKILL.md, scripts/, references/).
- Use the `writing-skills` skill to scaffold new capabilities discovered via this directory.

### 3. Updates
- Regularly check for updates to core skills (e.g., `ui-ux-pro-max`) and apply them to the workspace.
